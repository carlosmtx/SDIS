Argumentos Linha de Comandos

Se não forem especificados quaisquer parametros, o programa dá valores aos endereços IP e portos por defeito.
Caso pretenda utilizar valores para esses campos diferentes, o programa deve ser chamado da seguinte forma:
java Main <MCControlIP> <MCControlPort> <MCBackupIP> <MCBackupPort> <MCRecoveryIP> <MCRecoveryPort>

Definições
Através do menu definições o utilizador pode mudar os valores para o tamanho de chunk e replication degree.
